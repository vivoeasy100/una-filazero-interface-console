# 🚀 FilaZero Console

Sistema de login em **C# (Console Application)** desenvolvido para fins acadêmicos na disciplina de **Interação Humano-Computador (IHC)**.

---

## 📌 Sobre o Projeto

O **FilaZero Console** simula uma interface de login no terminal, aplicando conceitos de UX mesmo em aplicações de console.

O sistema permite:

* Entrada de email e senha
* Ocultação da senha com `*`
* Feedback visual de autenticação
* Simulação de carregamento (loading)
* Uso de cores para melhorar a experiência do usuário

---

## 🖥️ Demonstração

```bash
========================================
```

## ⚙️ Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/una-filazero-interface-console.git
```

### 2. Acesse a pasta

```bash
cd una-filazero-interface-console
```

### 3. Execute o projeto

```bash
dotnet run
```
